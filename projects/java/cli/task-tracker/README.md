Task CLI is a lightweight, cross-platform command-line task manager.
It works on Windows, Linux, and macOS with no dependencies other than Java.

How to install:
- Download the archive for your platform
    - task-cli-1.0.0.zip 
    - task-cli-1.0.0-release.tar.gz
- Extract the archive
- Open the README inside the extracted folder
- Follow the setup instructions provided there

Each archive contains:
- the task-cli.jar
- launcher script(s)
- platform-specific installation steps
